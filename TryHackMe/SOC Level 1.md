# SOC Level 1

The Security Operations Center (SOC) is a central hub for organizational security operations. SOC analysts work with logs, triage alerts, investigate suspicious activity, collaborate with teams, and respond to cyber incidents as the first line of defense.

This path focuses on building the technical and operational skills required for a Junior SOC Analyst role, covering areas such as:
- Log analysis
- Incident response
- SIEM platforms
- Threat detection
- Digital forensics
- Malware analysis
- Network monitoring
- Defensive security workflows
